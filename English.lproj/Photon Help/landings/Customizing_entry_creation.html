Customizing entry creation

Upon creating a new entry in most blogging packages you’ll be presented with an overwhelming number of options and text fields to fill out: entry title, entry body, entry excerpt, extended entry, keywords… you name it. Photon lets you fill in any and all of these fields with the data you have stored in your own iPhoto photo library; a set of preferences you can edit in the Entry Creation tab.
